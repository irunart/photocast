# PhotoCast frontend

- Staging: https://irunart.github.io/photocast/
- Prod: https://iest.run/photocast

```bash
# node 18 + 自带npm
npm install -g pnpm
pnpm i # 下载依赖包

# dev

pnpm dev # 启动开发服务器

# build 打包产物

pnpm build


```
